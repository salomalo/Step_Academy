css-селектори у jQuery

Умова

На початку кожного пункту цього списку додати кнопку, що викликає ф-ю, котра змінює вигляд елементів, що задовільняють умову, вказану в пункті.
Вибірку елементів здіснити jQuery css-селекторами лише всередині td#practice. Кожна кнопка змінює "свою" властивість(тло, рамка, текст ... ) також засобами jQuery
  1. всі <input> типу text додати червоне тло
  2. всі елементи, без id -- обвести лінією в крапочку
  3. лише <input> типу number без атрибуту value --
    додати синє тло
  4. всі елементи, у котрих id починається з message -- обвести рамкою
  5. всі елементи, з класом left у списку класів -- обвести заокругленою рамкою
  6. всі посилання на файли *.zip -- виділити жирним червоним текстом
  7. всі елементи з атрибутом data-action, що містить delete, як значення через пробіл
    зробити жовтими
  8. всі елементи з атрибутом data-action, що містить delete будь-де --
    зробити зеленими
  9. один елемент одразу за заголовком h3#widget-title на тому же рівні вкладеності --
    обвести бузковою рамкою
  10. всi <a>, після заголовку h3#widget-title на том же рівні вкладеності --
    зробити закреселеними
  11. <a> всередині останнього елемента списку #messages -- змінити колір тла
  12. кожен другий елемент списку #messages --
    зробити закреселеним
