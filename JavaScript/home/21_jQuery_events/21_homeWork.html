css-селектори у jQuery

Умова

На кожнен пункт цього списку повісити обробник, котрий змінює вигляд цільових елементів, що задовільняють умову, вказану в пункті.
Обробники "вішати" на різні події миші -- на кожен <li> обробник іншої події (рух миші, вхід і вихід миші, кліки лівою і правою кнопками, клік, подвійний клік і т.д.).
Коли подія завершилася, або сталася протилежна подія, має відновитися вигляд цільових елементів. Тобто, натиснули кнопку - щось стало червоним, відпустили -- червоний колір пропав; або двічі клікнули - щось стало червоним, ще раз двічі клікнули -- червоний колір пропав;
На <li>, котрий обробляє клік правою кнопкою не повинне з'являтися контекстне меню -- і лише на цьому <li>!
Вибірку елементів здіснити jQuery css-селекторами лише всередині td#practice.
В *.html має бути внесена лише одна зміна -- підключено .js файл з вашим кодом. Тобто не писати в html атрибути onclick чи ondoubleclick, а повісти обробники одразу після завершення завантаження документа.
  1. наведення курсора
    всі <input> типу text --
    додати червоне тло
  2. натиснення лывоъ клавиши (рызны властивосты зберегти)
    всі елементи, без id -- обвести лінією в крапочку
  3. натиснення правоъ клавиши
    лише <input> типу number без атрибуту value --
    додати синє тло
  4. проведення повз (рызны властивосты зберегти)
    всі елементи, у котрих id починається з message -- обвести рамкою
  5. подвийний клик
    всі елементи, з класом left у списку класів -- обвести заокругленою рамкою
  6. всі посилання на файли *.zip -- виділити жирним червоним текстом
  7. всі елементи з атрибутом data-action, що містить delete, як значення через пробіл
    зробити жовтими
  8. всі елементи з атрибутом data-action, що містить delete будь-де --
    зробити зеленими
  9. один елемент одразу за заголовком h3#widget-title на тому же рівні вкладеності --
    обвести бузковою рамкою
  10. всi <a>, після заголовку h3#widget-title на том же рівні вкладеності --
    зробити закреселеними
  11. <a> всередині останнього елемента списку #messages -- змінити колір тла
  12. наведення курсора
    кожен другий елемент списку #messages --
    зробити закреселеним
